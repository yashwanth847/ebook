Blender is a popular open source 3D modeling software used in advertising, animation, data visualization, physics simulation, photorealistic rendering, and more. Programmers can produce extremely complex and precise models that would be impossible to replicate by hand, while artists enjoy numerous new community-built add-ons.\nThe Blender Python API is an unparalleled programmable visualization environment. Using the API is made difficult due to its complex object hierarchy and vast documentation. Understanding the Blender Python API clearly explains the interface. You will become familiar with data structures and low-level concepts in both modeling and rendering with special attention given to optimizing procedurally generated models.
